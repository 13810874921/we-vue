## 开发

### 克隆代码

``` bash
git clone https://github.com/tianyong90/we-vue.git
```

### 安装依赖

``` bash
cd we-vue && npm install
```

> 建议使用 yarn

### 运行本地服务并开发

``` bash
npm run dev
```
> 完成以上步骤后将在浏览器里打开组件示例页面，修改代码保存后会自动完成热加载无需手动刷新

**开发过程中请勿进行构建(勿运行 build 相关命令)，代码请尽量规范，通过 Eslint 语法检查**
