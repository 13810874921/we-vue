## 安装

npm

```shell
npm install we-vue -S
```

yarn

```shell
yarn add we-vue -S
```

## 开发

1. 克隆代码

``` bash
git clone https://github.com/tianyong90/we-vue.git
```

2. 安装依赖

``` bash
cd we-vue && npm install
```

3. 运行本地服务并开发

``` bash
npm run dev
```
