# we-vue

we-vue 结合 vue2.0 强大的组件化机制和 weui 的微信官方视觉样式，形成了一套灵活的组件，适用于微信公众号等移动端开发。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
